Free & Purchasable Books, Courses, & Articles

Below is a comprehensive guide of all the resources I have collected that has helped me learn software programming, computer science, and web development. It's not 100% complete (I don't think it ever will be) but check back often for updates. If you have any recommendations on a resource that should be here, please tweet me.

Most of the resources below are free but others must be purchased. The free resources are labeled as free. I have provided links to where you can purchase the others.

Foundations of Computer Science

If you want to write and talk computer, you must have the foundational knowledge of computer science. This knowledge is vital because when you confront complex problems you'll need to know how to solve them.

  • Programming Methodology- free Youtube course by Professor Mehran Sahami at Stanford University for CS106A: Programming Methodology. The course is an Introduction to the engineering of computer applications emphasizing modern software engineering principles: object-oriented design, decomposition, encapsulation, abstraction, and testing.
  • Programming Abstractions- free Youtube course by Professor Julie Zelenski at Stanford University for CS106B: Programming Abstractions in C++. This course uses the C++ language to teach the material but it's not a course about the language but rather topics such as recursion, algorithmic analysis, and data abstraction.
  • Programming Paradigms- free Youtube course by Professor Jerry Cain at Stanford University for CS107: Programming Paradigms. The course aims to teach students how to write code for each of these individual languages and to understand the programming paradigms behind these languages.
  • MIT Open Courseware- Computer Science- MIT's free online computer science courses that covers introductory to advanced courses in computer science.
    • Introduction to Computer Science and Programming at MIT- free Youtube course by Professor Guttag covered in Python. Professor Guttag's course textbook is called Introduction to Computation and Programming Using Python
  • Problem Solving with Algorithms and Data Structures- free online textbook by Bradley Miller and David Ranum that covers algorithms and data structures in Python.
  • Data Structures and Algorithmic Thinking with Python- book by Narasimha Karumanchi that has 4.5 out of 5 stars with 61 reviews on Amazon. Understanding data structures and algorithms will help you develop efficient and effective solutions to problems.
  • Introduction to Algorithms- although this book isn't in Python it's the most recommended book on algorithms.
    • Introduction to Algorithms- free Youtube video course at MIT based on the book above.

Command Line, Git & Github

Git is a verson control system and Github is a web-based Git repository hosting service. While Git is a command line tool, GitHub provides a graphical interface. Basically you use Git to track, add and revert code changes. Github is where you'd host these changes. Github and Git backs up your code with the power of viewing, editing, adding, removing, and collaborating on your own or your team's past, present and future code.

  • Learn Command Line the Hard Way- free online textbook by Zed Shaw
  • Pro Git- open source online book. Probably my favorite resource for learning Git.
  • Git Documentation
  • Git Guide- a simple git guide by Roger Dudler. It doesn't cover everything but it's a quick tutorial to get you on the right track.
  • Github Documentation

Python

Learning Python resources

  • Dive Into Python 3- free online textbook by Mark Pilgrim. I loved his approach in this book.
  • Starting Out in Python- book by Tony Gaddis
  • Learning Python- deep introduction into the Python language. This book is huge but it covers it all. It will take a long time to finish but it's worth it.
  • Fluent Python- book by Luciano Ramalho. If you find yourself asking, "I've learned the basics of Python where do I go next?" If you haven't read Learning Python, this book should be next.
  • Effective Python- book by Brett Slatkin based on the 'Effective LANGUAGE-HERE' series but in Python. If you think you write good Python, give this book a try. I bet it will make you write better Pythonic code.
  • Think Python- free online book by Allen Downey. I have yet to read this one, partly because I've read the others above, but the title "Think Python: How to Think Like a Computer Scientist" implies it all. Since your coding in the language of computers it's important to think like a computer. With that in mind you'll write great code.
  • Python 3 Object Oriented Programming- Python is an Object-Oriented Programming language (OOP). It's important to implement this paradigm or your code can get messy.

Client/Frontend Web Development- HTML, CSS, Javascript, & Design

If you're going to build an application/program that the user must interact with then you're going to have to learn client/frontend (the user) development. HTML, CSS, and Javascript (JS) are the foundational languages for this. All websites use these three languages.

  • HTTP Protocol- free guides from Mozilla explaining the HTTP protocol- the foundation of the internet. It is required to know this before jumping into web development.
  • Learning Web Development- free guides from Mozilla about the fundamentals of web development- HTML, CSS and Javascript.
  • HTML and CSS: Design and Build Websites- a book from Jon Ducket. 884 reviews 4.5 out of 5 stars on Amazon. The layout and visuals in this book are well done. It makes it easy to learn the fundamentals of HTML and CSS.
  • HTML & CSS- free course from Codecademy
  • HTML course free course from W3Schools.com
  • AListApart.com- is a blogging website for people who make websites. A List Apart explores the design, development, and meaning of web content.
  • Javascript libraries and frameworks:
    • jQuery- a Javascript library that makes writing Javascript code easier by writing less. jQuery is a little old but it's an easy implementation to add behavior to user interfaces. Take it a step further and learn the React library or a framework below.
    • React- a Javascript library built by Facebook that uses the concepts of components and virtual DOM to simplify the process of building user interfaces.
    • Besides these two libraries there are frameworks such as Angular.js, Ember.js, and Backbone.js.

Databases- SQL, SQLAlchemy, PostgreSQL

When your application has to save data such as user's information (i.e., name, address, etc.), game information (i.e., wins, losses, points, money, etc), you use a database. SQL stands for Structured Query Language and is used to manage this data in a relational database management system (RDBMS) such as PostgreSQL, MySQL, Oracle, and more.

You'd use SQLAlchemy as a wrapper around the RDBMS to use Python classes and methods to represent SQL tables and queries. SQLAlchemy makes it easier to interact with the data in the RDBMS.

  • Learn SQL the Hard Way free online textbook by Zed Shaw
  • PostgreSQL Documentation- a relational database
  • SQLAlchemy Documentation- an object relation mapper for working with databases and Python.

Deployment

Once you've got an application that you'd like to serve to the world it's time to deploy it. But how? Look below.

  • Fullstackpython.com- a complete resource on the necessary skills and technologies to be a well-rounded Python developer.
  • Full Stack Python Guide to Deployment by Matthew Makai. The go-to resource to get your Python application built, deployed and managed in the cloud.
  • Amazon Web Services Documentation- Although these docs feel like you're lost in the Amazon (pun intended), they are really good at giving you the information you're looking for. If you use AWS with your Python app, you're most likely going to use Boto3- the AWS SDK for Python
  • Amazon Web Services in Action- a book by Andreas Wittig that is a great introduction to AWS. You'll learn about the best practices regarding security, high availability and scalability.

If you have an application that you'd like deployed quickly without the hassle of setting up servers then Heroku or AWS's Elastic Beanstalk may be for you.

  • Elastic Beanstalk
  • Heroku Documentation
