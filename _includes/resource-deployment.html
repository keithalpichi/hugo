Deployment

Once you've got an application that you'd like to serve to the world it's time to deploy it. But how? See below.

Resource Description Free?
Fullstackpython.com A complete resource on the necessary skills and technologies to be a well-rounded Python developer. Yes
Full Stack Python Guide to Deployment Book by Matthew Makai. The go-to resource to get your Python application built, deployed and managed in the cloud. No
Amazon Web Services Documentation Although these docs feel like you're lost in the Amazon (pun intended), they are really extensive. If you use AWS with your Python app, you're most likely going to use Boto3- the AWS SDK for Python Yes
Amazon Web Services in Action A book by Andreas Wittig that is a great introduction to AWS. You'll learn about the best practices regarding security, high availability and scalability. No

If you have an application that you'd like deployed quickly without the hassle of setting up servers then Heroku or AWS's Elastic Beanstalk may be for you.
