Text Editors

What program should you code in? There are many text editors out there but I recommend the following.

Text Editor Description Free?
Atom An easily hackable (customizable) text editor built by Github with a built-in package manager so you can quickly install packages to your liking. I currently use Atom and recommend it. I even made a theme for it Yes
Sublime Text Sublime has been around before Atom has. They share similar features but Sublime has established itself in the industry as the most popular. It's also known to be faster than Atom. Yes
Vim An upgrade to the old school 'vi' text editor. The biggest advantage of using Vim is that it's designed for use both from the command-line (Terminal) and as a standalone GUI text editor (so you don't have to switch between the two) and the ability to code fast and effectively by using keyboard shortcuts instead of using the mouse. Although it's fun to use and makes you a more efficient typer I'd recommend starting with one of the text editors above; especially if you're a beginner. Yes
