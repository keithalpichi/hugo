Client/Front-end Web Development- HTML, CSS, Javascript, & Design

If you're going to build an application/program that the user must interact with then you're going to have to learn client/frontend (the user) development. HTML, CSS, and Javascript (JS) are the foundational languages for this. All websites use these three languages.

Resource Description Free?
HTTP Protocol Guides from Mozilla explaining the HTTP protocol- the foundation of the internet. It is required to know this before jumping into web development. Yes
Learning Web Development Guides from Mozilla about the fundamentals of web development- HTML, CSS and Javascript. Yes
HTML and CSS: Design and Build Websites a book from Jon Ducket. 884 reviews 4.5 out of 5 stars on Amazon. The layout and visuals in this book are well done. It makes it easy to learn the fundamentals of HTML and CSS. No
AListApart.com A blogging website for people who make websites. A List Apart explores the design, development, and meaning of web content. Yes

Javascript libraries and frameworks:

Resource Description Free?
jQuery A Javascript library that makes writing Javascript code easier by writing less. jQuery is a little old but it's an easy implementation to add behavior to user interfaces. Take it a step further and learn the React library or a framework below. Yes
React A Javascript library built by Facebook that uses the concepts of components and virtual DOM to simplify the process of building user interfaces. Yes
Other Besides these two libraries there are frameworks such as Angular.js, Ember.js, and Backbone.js. Yes
